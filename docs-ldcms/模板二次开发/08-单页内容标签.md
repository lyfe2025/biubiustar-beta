# 单页内容标签

> 用于获取指定文档的详细内容，对应后台内容管理的内容

## 基本使用方法

```html
{ld:content cid="53" }
<!-- 内容描述 -->
<div>{$content.seo_description}</div>
<a href="{$content.url}">
    <span>查看详情 +</span>
</a>
{/ld:content}
```

## 调用参数

| 参数 | 是否必填 | 说明 |
|------|----------|------|
| cid | 是 | 所属栏目的ID |

## 常用字段

> 如需使用其他的字段请至后台内容管理下查找对应字段

| 字段名 | 说明 |
|--------|------|
| `{$content.url}` | 链接 |
| `{$content.title}` | 标题 |
| `{$content.image}` | 缩略图 |
| `{$content.seo_description}` | 描述 |
| `{$content.visits}` | 浏览次数 |
| `{$content.show_time}` | 发布时间 |

## 实际应用示例

```html
<div class="about-section">
    {ld:content cid="53"}
    <div class="about-content">

        {/if}

            {$content.content|raw}
        </div>

</div>
```

### 7. 使用技巧

#### 条件显示内容
```html
{ld:content cid="5"}
{if $content.title}
<div class="content-block">
    <h3>{$content.title}</h3>
    {if $content.image}
    <img src="{$content.image}" alt="{$content.title}">
    {/if}
    <p>{$content.seo_description}</p>
</div>
{/if}
{/ld:content}
```

#### 内容截取
```html
{ld:content cid="3"}
<div class="brief-intro">
    <h4>{$content.title}</h4>
    <p>{$content.seo_description|mb_substr=0,100,'utf-8'}...</p>
    <a href="{$content.url}">阅读全文</a>
</div>
{/ld:content}
```

#### 多个单页内容组合
```html
<div class="multi-content">
    {ld:content cid="11"}
    <div class="content-item">
        <h3>{$content.title}</h3>
        <p>{$content.seo_description}</p>
    </div>
    {/ld:content}
    
    {ld:content cid="12"}
    <div class="content-item">
        <h3>{$content.title}</h3>
        <p>{$content.seo_description}</p>
    </div>
    {/ld:content}
</div>
```
