# 友情链接标签

> 获取网站的友情链接，对应后台的"友情链接"
> 使用方式与轮播标签类似

## 基本使用方法

```html
{ld:link name='link'}
<a href="{$item.url}" target="{$item.target}">
    {$item.title}
</a>
{/ld:link}
```

## 调用参数

| 参数 | 是否必填 | 说明 |
|------|----------|------|
| name | 是 | 填写后台对应的友情链接类型 |
| limit | 否 | 调用数量 |
| alias | 否 | 默认为"item",代表循环列表的变量名称。 |
| empty | 否 | 数列为空时显示的数据 |
| key | 否 | 列表循环键名变量,默认为i |
| mod | 否 | 取模值，常用于控制换行，默认为2 |

## 常用字段

> 默认为`{$item.id}` 如果`alias`修改别名为'vo', 那么则变成`{$vo.id}`

| 字段名 | 说明 |
|--------|------|
| `{$item.url}` | 链接 |
| `{$item.target}` | 跳转方式 |
| `{$item.title}` | 标题 |
| `{$item.image}` | 图片 |

## 实际应用示例

### 1. 文字友情链接
```html
<div class="friend-links">
    <h3>友情链接</h3>
    <div class="link-list">
        {ld:link name="text_links"}
        <a href="{$item.url}" target="{$item.target}" title="{$item.title}">
            {$item.title}
        </a>
        {if $i % 6 == 0}<br>{/if}
        {/ld:link}
    </div>
</div>
```

### 2. 图片友情链接
```html
<div class="partner-logos">
    <h3>合作伙伴</h3>
    <div class="logo-grid">
        {ld:link name="logo_links"}
        <div class="logo-item">
            <a href="{$item.url}" target="{$item.target}" title="{$item.title}">
                <img src="{$item.image}" alt="{$item.title}">
            </a>
        </div>
        {/ld:link}
    </div>
</div>
```

### 3. 混合型友情链接
```html
<div class="friend-links-section">
    <div class="container">
        <h2>友情链接</h2>
        
        <!-- 图片链接 -->
        <div class="image-links">
            <h4>合作伙伴</h4>
            <div class="image-link-grid">
                {ld:link name="partner"}
                <a href="{$item.url}" target="{$item.target}" class="image-link">
                    <img src="{$item.image}" alt="{$item.title}" title="{$item.title}">
                </a>
                {/ld:link}
            </div>
        </div>
        
        <!-- 文字链接 -->
        <div class="text-links">
            <h4>友好网站</h4>
            <div class="text-link-list">
                {ld:link name="friends"}
                <a href="{$item.url}" target="{$item.target}" title="{$item.title}">
                    {$item.title}
                </a>
                {/ld:link}
            </div>
        </div>
    </div>
</div>
```

### 4. 轮播式友情链接
```html
<div class="partner-carousel">
    <h3>合作伙伴</h3>
    <div class="swiper-container partner-swiper">
        <div class="swiper-wrapper">
            {ld:link name="partners"}
            <div class="swiper-slide">
                <a href="{$item.url}" target="{$item.target}">
                    <img src="{$item.image}" alt="{$item.title}">
                    <span class="partner-name">{$item.title}</span>
                </a>
            </div>
            {/ld:link}
        </div>
        <div class="swiper-pagination"></div>
    </div>
</div>
```

### 5. 分类友情链接
```html
<div class="links-by-category">
    <!-- 政府网站 -->
    <div class="link-category">
        <h4>政府网站</h4>
        <ul class="link-list">
            {ld:link name="government"}
            <li>
                <a href="{$item.url}" target="{$item.target}" title="{$item.title}">
                    {$item.title}
                </a>
            </li>
            {/ld:link}
        </ul>
    </div>
    
    <!-- 行业网站 -->
    <div class="link-category">
        <h4>行业网站</h4>
        <ul class="link-list">
            {ld:link name="industry"}
            <li>
                <a href="{$item.url}" target="{$item.target}" title="{$item.title}">
                    {$item.title}
                </a>
            </li>
            {/ld:link}
        </ul>
    </div>
</div>
```

### 6. 底部友情链接
```html
<footer class="site-footer">
    <div class="container">
        <div class="footer-links">
            <div class="friend-links">
                <span class="links-label">友情链接：</span>
                {ld:link name="footer_links" limit="10"}
                <a href="{$item.url}" target="{$item.target}" rel="nofollow">
                    {$item.title}
                </a>
                {if !$last} | {/if}
                {/ld:link}
            </div>
        </div>
    </div>
</footer>
```

### 7. 卡片式友情链接
```html
<div class="partner-cards">
    <div class="row">
        {ld:link name="partners"}
        <div class="col-md-3 col-sm-6">
            <div class="partner-card">
                <a href="{$item.url}" target="{$item.target}">
                    <div class="partner-logo">
                        <img src="{$item.image}" alt="{$item.title}">
                    </div>
                    <div class="partner-info">
                        <h5>{$item.title}</h5>
                        <p class="partner-desc">{$item.description}</p>
                    </div>
                </a>
            </div>
        </div>
        {/ld:link}
    </div>
</div>
```

### 8. 响应式友情链接网格
```html
<div class="responsive-links">
    <h3>友情链接</h3>
    <div class="links-grid">
        {ld:link name="friends"}
        <div class="link-item">
            <a href="{$item.url}" target="{$item.target}">
                {if $item.image}
                <img src="{$item.image}" alt="{$item.title}">
                {else}
                <span class="text-link">{$item.title}</span>
                {/if}
            </a>
        </div>
        {/ld:link}
    </div>
</div>
```
