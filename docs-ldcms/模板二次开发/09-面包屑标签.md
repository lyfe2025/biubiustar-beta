# 面包屑标签

> 获取网页的面包屑导航，该标签不支持在首页使用

## 基本使用方法

```html
{ld:position cid="$category['id']"} /
<a href="{$item.url}">{$item.name}</a>
{/ld:position}
```

## 调用参数

| 参数 | 是否必填 | 说明 |
|------|----------|------|
| cid | 是 | 所属栏目的ID |
| alias | 否 | 默认为"item",代表循环列表的变量名称。 |
| empty | 否 | 数列为空时显示的数据 |
| key | 否 | 列表循环键名变量,默认为i |
| mod | 否 | 取模值，常用于控制换行，默认为2 |

## 常用字段

> 默认为`{$item.id}` 如果`alias`修改别名为'vo', 那么则变成`{$vo.id}`

| 字段名 | 说明 |
|--------|------|
| `{$item.url}` | 链接 |
| `{$item.name}` | 名称 |

## 实际应用示例

### 1. 标准面包屑导航
```html
<div class="breadcrumb">
    <a href="/">首页</a>
    {ld:position cid="$category['id']"}
    <span class="separator"> > </span>
    <a href="{$item.url}">{$item.name}</a>
    {/ld:position}
    <span class="separator"> > </span>
    <span class="current">{$category.name}</span>
</div>
```

### 2. Bootstrap样式面包屑
```html
<nav aria-label="breadcrumb">
    <ol class="breadcrumb">
        <li class="breadcrumb-item">
            <a href="/">
                <i class="fas fa-home"></i> 首页
            </a>
        </li>
        {ld:position cid="$category['id']"}
        <li class="breadcrumb-item">
            <a href="{$item.url}">{$item.name}</a>
        </li>
        {/ld:position}
        <li class="breadcrumb-item active" aria-current="page">
            {$category.name}
        </li>
    </ol>
</nav>
```

### 3. 详情页面包屑
```html
<div class="page-breadcrumb">
    <div class="container">
        <div class="breadcrumb-wrapper">
            <a href="/" class="home-link">
                <i class="icon-home"></i>
                <span>首页</span>
            </a>
            {ld:position cid="$content.cid"}
            <span class="divider">/</span>
            <a href="{$item.url}" class="category-link">{$item.name}</a>
            {/ld:position}
            <span class="divider">/</span>
            <span class="current-page">{$content.title}</span>
        </div>
    </div>
</div>
```

### 4. 移动端面包屑
```html
<div class="mobile-breadcrumb">
    <div class="breadcrumb-scroll">
        <a href="/" class="breadcrumb-item">首页</a>
        {ld:position cid="$category['id']"}
        <a href="{$item.url}" class="breadcrumb-item">{$item.name}</a>
        {/ld:position}
        <span class="breadcrumb-item current">{$category.name}</span>
    </div>
</div>
```

### 5. 带图标的面包屑
```html
<div class="breadcrumb-nav">
    <ul class="breadcrumb-list">
        <li class="breadcrumb-item">
            <a href="/">
                <i class="fas fa-home"></i>
                首页
            </a>
        </li>
        {ld:position cid="$category['id']"}
        <li class="breadcrumb-item">
            <i class="fas fa-chevron-right"></i>
            <a href="{$item.url}">{$item.name}</a>
        </li>
        {/ld:position}
        <li class="breadcrumb-item current">
            <i class="fas fa-chevron-right"></i>
            {$category.name}
        </li>
    </ul>
</div>
```

### 6. JSON-LD结构化数据面包屑
```html
<script type="application/ld+json">
{
    "@context": "https://schema.org",
    "@type": "BreadcrumbList",
    "itemListElement": [
        {
            "@type": "ListItem",
            "position": 1,
            "name": "首页",
            "item": "{$Request.domain}"
        },
        {ld:position cid="$category['id']"}
        {
            "@type": "ListItem",
            "position": {$i+1},
            "name": "{$item.name}",
            "item": "{$Request.domain}{$item.url}"
        },
        {/ld:position}
        {
            "@type": "ListItem",
            "position": {$i+2},
            "name": "{$category.name}",
            "item": "{$Request.domain}{$category.url}"
        }
    ]
}
</script>

<!-- 可见的面包屑 -->
<div class="breadcrumb">
    <a href="/">首页</a>
    {ld:position cid="$category['id']"}
    <span> > </span>
    <a href="{$item.url}">{$item.name}</a>
    {/ld:position}
    <span> > </span>
    <span>{$category.name}</span>
</div>
```

### 7. 简洁版面包屑
```html
<div class="simple-breadcrumb">
    {ld:position cid="$category['id']"}
    <a href="{$item.url}">{$item.name}</a> / 
    {/ld:position}
    <span class="current">{$category.name}</span>
</div>
```

### 8. 自定义分隔符面包屑
```html
<div class="custom-breadcrumb">
    <a href="/" class="breadcrumb-home">首页</a>
    {ld:position cid="$category['id']"}
    <span class="breadcrumb-separator">»</span>
    <a href="{$item.url}" class="breadcrumb-link">{$item.name}</a>
    {/ld:position}
    <span class="breadcrumb-separator">»</span>
    <span class="breadcrumb-current">{$category.name}</span>
</div>
```

## CSS样式参考

```css
/* 基础面包屑样式 */
.breadcrumb {
    padding: 10px 0;
    margin-bottom: 20px;
    font-size: 14px;
    color: #666;
}

.breadcrumb a {
    color: #007bff;
    text-decoration: none;
}

.breadcrumb a:hover {
    color: #0056b3;
    text-decoration: underline;
}

.breadcrumb .separator {
    margin: 0 8px;
    color: #999;
}

.breadcrumb .current {
    color: #333;
    font-weight: 500;
}

/* 移动端适配 */
@media (max-width: 768px) {
    .breadcrumb {
        font-size: 12px;
        padding: 8px 0;
    }
    
    .breadcrumb .separator {
        margin: 0 4px;
    }
}

/* Bootstrap风格 */
.breadcrumb {
    display: flex;
    flex-wrap: wrap;
    padding: 0.75rem 1rem;
    margin-bottom: 1rem;
    list-style: none;
    background-color: #e9ecef;
    border-radius: 0.25rem;
}

.breadcrumb-item + .breadcrumb-item::before {
    content: "/";
    display: inline-block;
    padding-right: 0.5rem;
    padding-left: 0.5rem;
    color: #6c757d;
}
```
